<h1 align="center">Site Pessoal</h1>

Repositório com o código-fonte do meu site pessoal e portfólio profissional.

O objetivo do site é apresentar minha experiência como desenvolvedor de software e meus projetos, de forma simples, clara e direta.

**Site publicado:**  
https://rodrigomaia.dev


## Sobre o projeto

O site foi desenvolvido com foco em clareza de informação, leitura rápida, estrutura limpa e baixa complexidade técnica.
A ideia é que o conteúdo seja o principal destaque, evitando dependência excessiva de frameworks.

O mesmo layout também serve como base para a geração do meu currículo em PDF, que está neste repositório e disponível para download no site.


## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (mínimo, quando necessário)
- Hospedagem estática
